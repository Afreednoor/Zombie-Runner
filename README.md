# Zombie Runner

A browser-playable survivor-like prototype built in Unity.

## Controls

- WASD - Move
- Auto Attack - Automatic
- Collect XP to level up
- Choose upgrades to survive longer

## Features

- Auto attacking combat
- Enemy waves
- XP & Level system
- Upgrade selection
- Adaptive world mutations
- Difficulty scaling
- Survival timer
- Kill counter
- Game Over system
